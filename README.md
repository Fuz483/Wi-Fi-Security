# Программное обеспечение для обнаружения и анализа сетевых угроз (Wifi-Security)
WiFi Security — это программный комплекс для мониторинга сетевой активности, обнаружения аномалий и анализа киберугроз в режиме реального времени. Проект сочетает в себе инструменты захвата пакетов, математический анализ трафика и встроенный прокси-сервер для безопасной инспекции данных.

Проект был разработан в рамках исследовательской работы для республиканского конкурса учащихся (Беларусь).

## Технологии
- Backend: Python 3.x, Flask, Flask-SocketIO.
- Анализ данных: Pandas, NumPy, Scapy (для работы с сетевыми пакетами).
- Frontend: HTML5, CSS3 (Modern Dark UI), JavaScript (Socket.IO, FontAwesome).
- Математические модели: Z-Score для аномалий, Энтропия Шеннона для анализа строк.

## Разработка

### Требования
Для работы захвата пакетов (sniffing) в Windows требуется установленный [Npcap](https://npcap.com/) или WinPcap.

### Установка зависимостей
Для клонирования, выполните команду:
```sh
git clone https://github.com/your-username/wifi-security-hub.git
cd wifi-security-hub
```
Для установки зависимостей, выполните команду:
```sh
pip install flask flask-socketio scapy pandas numpy requests
```

## Запуск
Для корректной работы захвата сетевого трафика скрипт необходимо запускать от имени администратора:
```sh
python app.py
```
## Запуск
- Мониторинг: Нажмите "Запустить сканирование" для анализа текущего сетевого интерфейса.
- Анализ датасетов: Загрузите файлы .pcap или .csv для пост-анализа зафиксированных атак.
- Проверка URL: Вставьте подозрительную ссылку в поле анализатора для мгновенной оценки риска.
- Прокси: Активируйте встроенный прокси-сервер для контроля внешних соединений.

### Зачем вы разработали этот проект?
Разработка комплексного программного обеспечения для мониторинга сетевой активности, способного в режиме реального времени обнаруживать типичные киберугрозы в Wi-Fi сетях и предоставлять пользователю наглядную оценку уровня безопасности соединения.
